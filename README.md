# LLM-Powered-Dynamic-Pricing-Engine-with-Causal-Inference-for-Retail-and-E-Commerce
An AI-powered dynamic pricing engine that uses causal inference to understand why demand changes and recommends optimal pricing strategies. By combining machine learning, Bayesian optimization, and LLM-generated insights, the system helps businesses make smarter, explainable, and profit-focused pricing decisions.
## Tech Stack
- Frontend: React + Vite + Tailwind CSS
- Backend: FastAPI (Python)
- Database: MySQL
- AI/ML: DoWhy, EconML, CausalML, LangChain
- LLM: GPT-4 / Claude API
